About
Search
Type in the name of any movie or show and see how it'll look. Be warned that clients use various sizes and rounding, you may have to tweak to your own client.
TMDB Key
Required for poster art, logos, and metadata. If the server host has set this, you don't need it.
MDBlist Key
Required for ratings, awards, and keyword data. If the server host has set this you don't need it.
Native Logo Language
The language Posters+ requests title logos in. If it's unavailable it falls back to English.
Logo Priority
Order logos are picked in. "Native" is your Native Logo Language above; "Original" is the content's own language (e.g. Japanese for anime); "Text" renders the translated title as text. Choose whether your language or the content's original language wins, or restrict to your language only.
Top Vignette
A dark fade across the top of the poster that keeps badges and age ratings readable over bright art. Four levels: Off / Low (subtle), Medium (balanced), High (strongest — the legacy look). Lower presets work better on art that's already dark up top.
Bottom Vignette
Same four-level scale as the Top Vignette but for the bottom of the poster. Mainly useful if you've hidden the rating label and don't need the dark fade behind it — drop to Low or Off to let the art breathe. Use High (default) or Medium when you have a Score Bar / Clean / Minimalist / Compact label down there.
Display Modes
None
Clean art, no score overlay. Best for a purely textless aesthetic.
Score Bar
Horizontal gradient bar with a percentage fill based on the rating. Under 50, 50 to 69, 70 to 84 and 85+ are the color tiers. Six band uses six tiers, instead.
Clean
Genre ★ score as centred text near the bottom. Omits year and the score bar.
Minimalist
Small render in the bottom right, the score seperator uses the same mechanics as score bar as a solid color rather than fill. The bottom vignette is also intentionally weaker in this mode.
Compact
"All info in one strip" — Genre · Year · Info sash text, centred near the bottom. The whole line (text and both dot separators) is painted in the rating-derived colour from the chosen palette, so the line itself doubles as the score signal. Falls back to silver if no rating is available. Toggle Show Year off if you'd rather drop the year and use the freed-up space for a larger font. Compact never disables other layers — keep the diagonal sash or quality badges too if you want.
Score Colour Palettes
Light
Designed to only use the brighest versions of each color, for "at a glance" ratings.
Dark/light
Dark red, red, amber, yellow, dark green, bright green to stick with easy to understand colors.
Prestige
Focuses on the bronze, silver and gold color scheme used throughout Posters+.
Glow & Layout
  • Threshold determines when glow activates, blur increases the radius and alpha increases the brightness.
  • Font size and vertical position are intended for customization across many clients.
  • In Minimalist mode, horizontal offset moves the pip away from the edge to improve client compatibility.
Preservation Mode

If you want to leave as much of the original art untouched as possible, combine:

  • Rating display → Compact (small single line at the bottom, score-coloured)
  • Core Config → Top Vignette → Off or Low
  • Quality Badges → Hidden (or stick with Quality Notch for a small accent)

This is the most art-respecting configuration — no top-corner overlays, no top fade, just a small text strip at the bottom that doubles as the rating signal via colour.

Textless Mode
  • Textless — skips rendering the logo on the textless poster, use if you want to use your clients title display below the poster.
  • If no textless poster exists, the server falls back to cropping the title's landscape backdrop into a portrait frame. If neither exists, we fallback to the original art. If there's no art at all, a genre-tinted dark canvas is used to avoid 404.
Logo Sizing & Position
Max width
Logo is scaled so it never exceeds this fraction of the poster width.
Max height
Logo is scaled so it never exceeds this fraction of the poster height. Both constraints apply simultaneously — whichever is tighter wins.
Bottom ratio
Distance from the bottom edge as a fraction of poster height. Higher value moves the logo further up.
Sashes
  • Award wins and nominations will only show best picture, outstanding drama/comedy/limited series, and the five main Golden Globes to avoid spam.
  • Film festivals cover (Cannes, Sundance, Venice) and many more.
  • Trending on TMDB is refreshed daily and uses the US top 40.
  • Cult classic status and true story use Mdblist keywords.
  • Highlighted studios, directors and cast are based on your instance operators settings.
  • Newly streaming pulls from both official digital release dates and Reddit.
Display Modes
None
No badge. Cleanest look, no AIOStreams dependency.
Quality + Age
Colors the age rating based on quality. Uses a six point scoring system, 1080p, WebDL, HDR are 1. 4K, Remux, DV or HDR10+ are 2. Requires 2/4/6 for each tier.
Badge Row
PNG icon badges for format, badges are three categories: 1080p/4K, Web/Remux, HDR/DV/HDR10+
Age Only
Content-rating pill only — no quality dependency, works without AIOStreams.
Quality Notch
Very minimal accent notch, follows the same scoring system as quality + age.
Sizing & Positioning
  • Height sets badge image size in pixels. Mode 2 (Badge Row) defaults to 32px; other modes to 36px; Tier Accent to 20px.
  • Gap controls horizontal spacing between badges in Badge Row mode.
  • Anchor X / Y shift the badge cluster relative to the top-left corner of the poster.
  • Quality data is fetched from AIOStreams in the background on first request — the first render may show no badges while the cache warms. Subsequent requests will include badges.
How Weights Work
  • Each slider sets the contribution of that source's score to the final weighted average.
  • Set a source to 0 to exclude it entirely — it won't be used even if data exists.
  • Sources with no data for a title are skipped automatically; the remaining sources are re-weighted.
  • Weights don't need to sum to 1 — they are normalised before computing the score.
Source Notes
Letterboxd
Default strongest signal for film; community-curated, skews cinephile.
Trakt
Default strongest signal for TV; broad audience with good coverage of streaming originals.
RT Critic
Rotten Tomatoes critic score, tends to have high scores as they use pass/fail rather than numeric.
RT User
RT audience score, often diverges from critics.
IMDb
Wery wide coverage but can be skewed on polarising titles.
TMDB
Very wide coverage, fairly safe option.
Metacritic
Critic aggregate, below average coverage and skews lower scores.
Roger Ebert
Lowest coverage, as it's one singular review you can see scores of 100 here.
MyAnimeList
Anime only. Anime fans can set to 99% and set their movie/TV providers to around 1-2% and it'll normalize accordingly

All data is fetched via MDBlist — an MDBlist API key is required for scores to appear.

What's new in v1.0.0
  • Info Sash badge style — rounded rectangle alternative to the diagonal sash, with adjustable X/Y position
  • Per-section ⓘ info modals throughout the configurator
  • Golden Globe wins and nominations now have their own dedicated sash slots
  • Backdrop fallback — TMDB landscape backdrop is cropped to portrait when no textless poster (or no poster at all) is available
  • Genre-tinted fallback canvas — atmospheric colour matched to genre when no art exists
  • Smoother edges — Cairo antialiasing on award badge corners and pill renders
  • MDBlist rate-limit cooldown — automatic global pause on 429 to prevent cascade failures
